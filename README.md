# CustomAssetImporter ver1.0
This asset is useful to set asset's import settings automatically.
### Supported Unity Versions
Unity5.5 or newer

## Usage
Open setting window with
##### Window > CustomAssetImporter > [Audio/Texuture/Model] Importer
